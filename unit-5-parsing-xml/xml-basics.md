## XML Basics

XML can be set up in two different ways:

```XML
<library>
    <book title="Example Title" pages="999" />
    <book title="Another Title" pages="999" />
</library>
```

or

```
<library>
    <book>
        <title>Example Title</title>
        <pages>999</pages>
    </book>
    <book>
        <title>Another Title</title>
        <pages>999</pages>
    </book>
</library>
```

Elements are the tags used and attributes supply supplemental information though these lines are blurred when you compare the two examples.
